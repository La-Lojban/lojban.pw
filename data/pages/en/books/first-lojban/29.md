### What is a lerfu?

<speaker sprite="sev1">You already know that Lojban uses Latin letters. Back in Lesson 2, you learned how to _name_ them:

- **Consonant letter names:** `[consonant] + y + .` → **by.**, **cy.**, **sy.**, **zy.** …
- **Vowel letter names:** `. + [vowel] + bu` → **.abu**, **.ebu**, **.ibu**, **.obu**, **.ubu**

A word used as a letter name is called a **lerfu** (letter/symbol word).</speaker>

lerfu
: x<sub>1</sub> is a letter/symbol representing x<sub>2</sub> in alphabet/writing system x<sub>3</sub>

<speaker sprite="sor1">So **by.** is the Lojban name for the letter "b" and **.abu** is the name for "a". Got it. But I never heard about the period `.` or apostrophe `'`. What are those called?</speaker>

<speaker sprite="sev1">Good catch. They're special:

- **Period `.`** → **denpa bu** ("pause-letter")
- **Apostrophe `'`** → **.y'y.** (literally the cmavo **.y.** spoken as a lerfu — two glottal pauses around the schwa)</speaker>

denpa
: x<sub>1</sub> waits for x<sub>2</sub> (event) while doing x<sub>3</sub>, until x<sub>4</sub> begins

<speaker sprite="sor2">So **denpa bu** is literally the "waiting letter." A pause. That's kind of poetic.</speaker>

<speaker sprite="sev2">Also works for letters Lojban doesn't normally use, like **h**, **q**, or **w**. Since **'** already sounds like an "h", people often write **h** as **.y'y. bu** ("an h-like letter"). And **q** is typically **.ky. bu** ("a k-like letter"), since **q** in most languages sounds like **k**.</speaker>

### **bu**: turn any word into a letter-symbol

<speaker sprite="sev3">The real power tool here is **bu**. Stick **bu** after _almost_ any Lojban word, and it becomes a lerfu — a symbol standing for that word.

Common examples you'll see in practice:</speaker>

> **.ui bu** → 😊 (the happiness-letter)
> **.iu bu** → ❤️ (the love-letter)
> **denpa bu** → `.` (the pause-letter)

<speaker sprite="sor1">So **.ui bu** is literally the emoji **:)**? That's delightful. Can I put **bu** after _any_ Lojban word?</speaker>

<speaker sprite="sev3">Almost. A few words eat up whatever comes right after them before **bu** gets a chance to act:

- **zo**, **zoi**, **la'o**, **lo'u** — all of these grab the _next word_ or _next passage_ as a quotation before **bu** can form a lerfu.

So **zo bu** doesn't make a "zo-letter"; it quotes the word **bu** itself.</speaker>

<speaker sprite="sor5">Edge case, sure. I'll remember: quotation words come first.</speaker>

### A string of lerfu = one sumti

<speaker sprite="sev1">Here's something subtle. Consider this sentence:</speaker>

> **.abu dunda by. cy.**
> _A gives bc._

<speaker sprite="sor2">Wait — "a gives bc"? Not "a gives b to c"?</speaker>

<speaker sprite="sev4">Exactly. **by. cy.** is a _sequence_ of lerfu and sequences are treated as a single sumti — the string "bc". So the sentence has only _two_ arguments: the giver (a) and the thing given (bc).

Word spacing doesn't matter: **by.cy.** and **by. cy.** both mean the letter string "bc".</speaker>

<speaker sprite="sor7">So how do I say "a gives b to c"?</speaker>

<speaker sprite="sev1">Use **boi** to explicitly end the lerfu (or number) sequence:</speaker>

boi
: terminator for lerfu strings and number sequences

> **.abu dunda by. boi cy.**
> _a gives b to c._

<speaker sprite="sor2">**boi** cuts the lerfu string at that point. So **cy.** stays separate, filling x₃ of **dunda** as the recipient. Okay, I'll remember that one.</speaker>

<speaker sprite="sev2">**boi** works with numbers too. If you ever see something like **li pa by.** (the number 1 followed by the letter b), writing **li pa boi by.** keeps them from fusing into a hybrid string.</speaker>

### Acronym name words

<speaker sprite="sev2">Last topic: how to Lojbanize acronyms like NASA, FBI, CD, or BPFK.

The official recipe:</speaker>

1. Spell out each letter with its Lojban lerfu name.
2. If two vowel-name lerfu come next to each other, insert an apostrophe between them.
3. Add any consonant at the very end (the last consonant lerfu already in the string is convenient, or use **n**, or use the first letter of the source culture's word for the organisation).

> **la .ny'abusy'abus.** → NASA  
> **la .cydyd.** → CD  
> **la .fyby'ibun.** → FBI  
> **la .bypyfykyk.** → BPFK

<speaker sprite="sor1">So every letter is just spelled out in Lojban order, smooshed together, with a final consonant tacked on?</speaker>

<speaker sprite="sev2">Exactly. The last consonant makes it a proper cmevla (remember: name words must end in a consonant). As long as it's intelligible, you have some flexibility — some people drop all the **bu** parts and just use the Lojban letter names directly, others invent shortcuts. _Communicability_ is the real rule.</speaker>
